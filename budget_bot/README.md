# Budget Bot

Телеграм-бот для совместного учёта бюджета между несколькими пользователями.

## Возможности

- Создание категорий с месячным бюджетом
- Совместные траты между привязанными пользователями
- Уведомления всем участникам при добавлении траты
- Описание к каждой трате
- Автоматический сброс счётчиков 1-го числа каждого месяца
- Прогресс-бар расходов
- История трат по категории

## Запуск

```bash
# Установить зависимости
pip install -r requirements.txt

# Запустить бота
BOT_TOKEN=<ваш_токен> python main.py
```

## Команды

| Команда | Описание |
|---------|----------|
| `/start` | Начать работу с ботом |
| `/myid` | Показать ваш Telegram ID |
| `/link <ID>` | Отправить запрос привязки пользователю |
| `/accept <ID>` | Принять запрос привязки |
| `/addcat <название> <бюджет>` | Создать категорию с месячным бюджетом |
| `/setbudget <категория> <сумма>` | Изменить бюджет категории |
| `/spend <категория> <сумма> [описание]` | Добавить трату |
| `/budget` | Показать текущий бюджет по всем категориям |
| `/history <категория>` | История последних 10 трат категории |
| `/delcat <категория>` | Удалить категорию и все её траты |

## Пример использования

1. Оба пользователя пишут боту `/start`
2. Пользователь A узнаёт ID пользователя B через `/myid`
3. Пользователь A: `/link <ID_B>`
4. Пользователь B: `/accept <ID_A>`
5. Создать категорию: `/addcat Продукты 15000`
6. Пользователь A: `/spend Продукты 350 Молоко и хлеб`
7. Пользователь B получает уведомление с обновлённым балансом
8. `/budget` — посмотреть общий бюджет

## Структура базы данных

- `users` — пользователи (привязаны к группе)
- `groups` — группы пользователей с общим бюджетом
- `categories` — категории с месячным бюджетом
- `expenses` — траты с описанием и автором
- `link_requests` — запросы на привязку
