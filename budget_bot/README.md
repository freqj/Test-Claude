# Budget Bot

Telegram-бот для совместного учёта семейного или командного бюджета. Поддерживает категории трат, месячные лимиты, историю расходов, фото чеков и приватные категории.

## Возможности

- **Категории с бюджетом** — создавайте категории с месячным лимитом, общие для всех участников группы
- **Приватные категории** — категории, видимые только вам и не раскрываемые партнёру
- **Запись трат** — добавляйте суммы с описанием и фото чека
- **Прогресс-бар** — наглядно видно, сколько потрачено из лимита
- **Уведомления** — партнёр сразу узнаёт о новых тратах в общих категориях
- **История** — список всех расходов по категории за месяц
- **Совместный бюджет** — привяжите партнёра и ведите общий учёт
- **Сброс 1-го числа** — счётчики трат обнуляются автоматически каждый месяц

## Команды

| Команда | Описание |
|---|---|
| `/start` | Начало работы, список команд |
| `/addcat <название> <бюджет>` | Создать общую категорию с месячным бюджетом |
| `/addprivcat <название> <бюджет>` | Создать приватную категорию (видна только вам) |
| `/setbudget <категория> <сумма>` | Изменить бюджет категории |
| `/spend <категория> <сумма> [описание]` | Записать трату |
| `/budget` | Текущий бюджет по всем категориям |
| `/history <категория>` | История трат по категории |
| `/delcat <категория>` | Удалить категорию |
| `/link <ID>` | Пригласить партнёра в общий бюджет |
| `/accept <ID>` | Принять приглашение в общий бюджет |
| `/myid` | Узнать свой Telegram ID |
| `/cancel` | Отменить текущую операцию |

Все команды работают пошагово — достаточно ввести команду без аргументов и следовать подсказкам.

## Приватные категории

Категории, созданные через `/addprivcat`, видны только вам:
- не отображаются в бюджете партнёра
- не рассылают уведомления при записи трат
- помечены иконкой 🔒 в списках выбора

## Пример использования

1. Оба пользователя пишут боту `/start`
2. Пользователь A узнаёт свой ID через `/myid`
3. Пользователь B: `/link <ID_A>`
4. Пользователь A: `/accept <ID_B>`
5. Создать категорию: `/addcat Продукты 15000`
6. Пользователь B: `/spend Продукты 350 Молоко и хлеб`
7. Пользователь A получает уведомление с обновлённым балансом
8. `/budget` — посмотреть общий бюджет

## Установка и запуск

**Требования:** Python 3.11+

```bash
pip install -r requirements.txt
BOT_TOKEN=<ваш_токен> python main.py
```

Переменные окружения:

| Переменная | По умолчанию | Описание |
|---|---|---|
| `BOT_TOKEN` | — | Токен бота из [@BotFather](https://t.me/BotFather) (обязательно) |
| `DB_PATH` | `budget.db` рядом с `main.py` | Путь к файлу базы данных SQLite |

## Структура проекта

```
budget_bot/
├── main.py        # Инициализация бота, планировщик, регистрация хендлеров
├── handlers.py    # Все команды и ConversationHandler-ы
├── database.py    # Работа с SQLite (aiosqlite)
└── budget.db      # База данных (создаётся автоматически)
```

## Структура базы данных

SQLite с автоматическими миграциями при старте.

| Таблица | Описание |
|---|---|
| `users` | Пользователи (telegram_id, username, group_id) |
| `groups` | Группы совместного бюджета |
| `categories` | Категории (общие: `owner_user_id=0`, приватные: `owner_user_id=users.id`) |
| `expenses` | Записи трат (amount, description, photo_file_id) |
| `link_requests` | Ожидающие запросы на привязку партнёра |
